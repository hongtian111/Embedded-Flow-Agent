# Embedded-Flow-Agent
我构建了一个基于 OpenClaw 框架的嵌入式研发全栈 Agent，旨在解决硬件开发中技术手册冗长碎片化、底层驱动配置繁琐以及手动调试反馈周期长的核心痛点。该项目深度利用 MiMo 模型超长上下文能力，对 ASR Pro 等芯片的百万字级别 Datasheet 进行精准 RAG 检索，并通过长链条推理（CoT）自主生成引脚定义与外设初始化代码。其核心逻辑采用多 Agent 协作闭环：编码 Agent 负责逻辑实现，诊断 Agent 在 AutoDL 容器环境下实时解析编译日志与串口 Log，通过自我反思（Self-Correction）机制进行代码迭代与自动重构，实现了从“文档阅读”到“代码落地”的端到端自动化。
