# 嵌入流代理 (Embedded-Flow Agent)

这是一个基于 MiMo 大模型驱动的嵌入式研发全栈智能体，专门用于加速 ASR Pro、STM32 等硬件开发流程。

### 核心功能：
- **芯片手册深度解析**：利用大模型长上下文能力，秒级理解上千页的 Datasheet。
- **驱动代码自动生成**：从自然语言需求直接转化为可运行的底层 C 代码。
- **硬件调试自反思**：根据编译报错和串口 Log 自动进行逻辑修复与重构。

### 技术栈：
- 大模型：Xiaomi MiMo-V2.5-Pro
- 环境：AutoDL / Linux / Python 3.9+
